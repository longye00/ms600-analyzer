# MS600 多光谱作物分析工具

一个**纯前端、可离线运行**的单文件网页（`MS600.html`），用于把 MS600 / 多光谱相机拍摄的 6 波段 TIFF 照片，自动完成**辐射定标 → 波段配准 → 植被指数计算 → 冠层分割与统计**。打开 HTML 即用，照片不上传服务器，全部在本地浏览器内计算。

---

##  功能特性

- **两种分析模式**
  - **田间 / 无人机批量版**：用单独拍摄的灰板组做辐射定标，批量处理多组航拍照片。
  - **成像台 · 穴盘单株版**：逐张定标，适合室内成像台 / 穴盘单株表型。
- **6 波段自动对齐**：先全局亚像素平移，再做**分块局部配准（位移场扭曲）**，校正近景/景深差异造成的波段错位（彩色重影）。
- **辐射定标**：基于灰板反射率把 DN 值转换为反射率，自动跳过饱和波段。
- **多种植被指数**：NDVI、SAVI、OSAVI、GNDVI、CIgreen、NDRE、CIrededge、NDRE750、MCARI。
- **冠层分割**：NDVI + 近红外下限 + 近红外/红差值 + 绿色超量（ExG）多重判据，自动阈值（Otsu）并支持手动覆盖。
- **可视化输出**：假彩色图、自然色图、NDVI 热力图、植被掩膜叠加图，以及覆盖度与各指数的掩膜内统计。

---

##  文件命名要求

每张照片需按波段命名，程序据此识别分组与波段：

```
MS600_<组号>_<波段nm>.tif
```

- **波段（nm）**：`450`（蓝）、`555`（绿）、`660`（红）、`720`（红边）、`750`（红边）、`840`（近红外）
- **组号**：同一次拍摄的 6 张用同一组号，例如 `MS600_0003_450nm.tif … MS600_0003_840nm.tif`
- 田间模式下，灰板照片也按同样规则命名（例如灰板组 `0001`），与作物照片一起上传，并在「灰板组编号 grey_id」中填写对应组号。

---

##  使用步骤

1. 用浏览器打开 `MS600.html`。
2. 选择分析模式（田间批量 / 成像台单株）。
3. 一次性选入所有照片（含灰板组）。
4. 田间模式下，在「灰板组编号 grey_id」填入灰板那一组的组号。
5. 点击「开始分析」，查看假彩色、NDVI、植被掩膜与统计结果。

---

##  处理流程

1. **读取 TIFF** → 解析每组 6 个波段。
2. **灰板定标** → 在灰板上取中位 DN，结合参考反射率（默认 0.61）换算反射率；饱和波段自动剔除。
3. **波段配准** → 以红波段为基准，全局平移 + 分块局部位移场，对齐其余波段。
4. **指数计算** → 由反射率计算 NDVI 等指数。
5. **冠层分割** → 多判据阈值分割，去小连通域、填内部孔洞。
6. **统计与渲染** → 输出覆盖度、掩膜内指数均值/中位数，以及各类图像。

---

##  主要参数

| 参数 | 说明 |
|---|---|
| 灰板组编号 grey_id | 田间模式下灰板那一组的组号 |
| 灰板参考反射率 | 灰板标称反射率，默认 0.61 |
| 波段配准 | 全局平移 + 分块局部配准，校正近景错位 |
| NDVI 阈值 | 留空自动（Otsu），可手动指定 |
| NDVI 阈值下限 | 自动阈值的下限，默认 0.25 |
| 饱和阈值 | DN 超过该值的像素视为饱和，默认 64000 |
| 近红外下限 / 近红外-红差值 | 抑制土壤/背景误检 |
| 绿色超量下限 ExG | 区分绿叶与棕色基质；**紫/红叶作物请设为 `off`** |
| 最小连通域面积 | 去除细碎误检 |

##  精度与注意事项

- **灰板饱和**：450nm 在白板上可能接近/超过饱和阈值，会被自动剔除；建议适当压低曝光。
- **紫 / 红叶作物**：这类叶片 NDVI 虽高，但绿光不占优势，ExG（绿色超量）多为负，默认门限会把大片叶肉漏检。处理紫/红叶时请将「绿色超量下限 ExG」设为 `off`。
- **画幅边缘 / 顶部**：无波段重叠区域可能残留细条纹，属正常现象。
- **近景照片**：景深差异大时，分块局部配准可显著减少彩色重影，但极端视差仍可能有少量残留。

##  常见问题

- **提示「未找到灰板组」**：检查 grey_id 是否与灰板照片的组号一致，且灰板 6 个波段都已选入。
- **叶片有彩色重影 / 错位**：确认已开启「波段配准」；近景照片依赖分块局部配准。
- **植被掩膜漏检大片叶片**：多为紫/红叶导致 ExG 门限误剔，将「绿色超量下限 ExG」设为 `off` 重新分析。
