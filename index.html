使用说明与介绍

本工具用于处理 MS600 六波段多光谱相机（450 / 555 / 660 / 720 / 750 / 840 nm）拍摄的原始 TIFF，在浏览器内自动完成「波段配准 → 辐射定标 → 植被分割 → 反射率与植被指数统计」。全部计算在本地完成，照片不会上传到任何服务器，可离线使用。

文件命名要求

请上传形如 MS600_<编号>_<波长>nm.tif 的文件（例：MS600_0001_660nm.tif）。同一编号为一组，每组需集齐 450 / 555 / 660 / 720 / 750 / 840 nm 六个波段。支持未压缩与 LZW / PackBits 压缩的 8 / 16 / 32 位 TIFF。

操作步骤

1选择分析流程：按拍摄场景选「田间 / 无人机批量版」或「成像台 · 穴盘单株版」。
2上传照片：点击或拖拽 TIFF；界面会按编号显示每组波段是否齐全。田间版请连同那一组灰板照片一起上传。
3开始分析：如需微调（如田间版的「灰板组编号」）可展开「高级参数」，否则直接点「开始分析」。

处理流程

  • 波段配准：以 660 nm 为基准对齐其余波段，消除多镜头视差。采用「全局平移 + 分块局部配准」：先估计主体 ROI 内的亚像素全局位移，再逐块细调生成平滑位移场并双线性扭曲，可校正近景 / 景深差异带来的随位置变化的错位。
  • 辐射定标：用灰板 / 定标板将 DN 转为反射率，并裁剪到 [0, 1]；饱和波段自动排除。
  • 植被分割：NDVI 自适应阈值 + 近红外 / 红光差值约束 + 形态学清理 + 去小连通域，生成植被掩膜。
  • 指数统计：在掩膜内统计逐波段反射率与 NDVI / GNDVI / NDRE / CIrededge 等植被指数。

田间 / 无人机批量版（第一种方式）

一次拍摄多组照片、全程光照基本稳定。选一组灰板照片作为统一辐射基准，其余组用它定标。在「灰板组编号 grey_id」里填入那组灰板的编号（默认 0000），并把灰板照片连同作物照片一起上传。适合大田 / 冠层尺度表型。

成像台 · 穴盘单株版（第二种方式）

室内固定支架 + 可控光照，每张照片 = 1 个穴盘 + 1 块旁置定标板。逐张独立定标（每张自动检测定标板），从根本上消除照片间光照漂移；可按网格切到单穴。这种方式不需要单独的灰板组。

波段与植被指数

六个波段：

  • 450 nm 蓝 · 555 nm 绿 · 660 nm 红（绿 / 红为叶绿素吸收峰）
  • 720 / 750 nm 红边（对叶绿素含量、氮胁敏感）· 840 nm 近红外（反映冠层结构 / 生物量）

主要植被指数：

  • NDVI = (NIR−Red)/(NIR+Red)，绿度 / 长势综合指标。
  • GNDVI（用绿代替红）、SAVI / OSAVI（土壤调整，弱化裸土干扰）。
  • NDRE / CIrededge / MCARI（红边类，对叶绿素与肌理变化更敏感，适合中后期冠层）。

主要参数说明

  • 波段配准：默认开启（全局 + 分块局部）；若各波段本就对齐可关闭以加速。
  • 灰板组编号 grey_id（田间版）：填单独拍摄的那组灰板的编号（如 0001）。未填或填错会提示可选编号。
  • 灰板 / 定标板标称反射率：按实际板子的标称值填写（直接影响反射率绝对量）。
  • NDVI 阈值：auto 为 Otsu 自适应；也可填固定数值。
  • 阈值下限保护 min_thresh：防止自适应阈值过低让背景渗入，填 off 关闭。
  • 近红外反射率下限 R840 / NIR−Red 最小差值：抑制黑布、阴影等被误判为植被。
  • 单穴网格 行x列（穴盘版）：填写后可切分到单穴，与破坏取样一一对应。

输出结果

  • 逐组 / 整盘的反射率与植被指数统计表（可下载 CSV）。
  • 每组的 QC 假彩色图、植被掩膜、覆盖度与所用 NDVI 阈值。
  • 穴盘版开启网格时额外输出单穴级统计。
精度与注意事项

波段配准采用全局平移 + 分块局部配准（位移场扭曲）：航拍、近景特写都能处理，近景随景深变化的视差也能较好校正。仍建议对照输出的 QC 假彩色图检查边缘是否有彩色重影。几点物理限制：① 画幅最边缘 / 最顶部某些波段超出画外、无重叠区，可能残留条纹；② 灰板 / 定标板饱和的波段会被自动排除（拍灰板时适当压低曝光）；③ 红 / 紫叶品种红光反射偏高，NDVI 天生偏低、覆盖度偏保守，属植物特性而非算法错误。

常见问题

  • 某编号没结果？ 多为六波段不全，请检查文件命名与缺失波段提示。
  • 提示“未找到灰板组”？（田间版）说明本次没上传灰板组，或「灰板组编号」填错。请把灰板照片一起上传并填对编号；若本批不含灰板，请改用“穴盘单株”方式。
  • 近景 / 特写照叶片有彩色重影？ 现已由分块局部配准校正；若仍有残留，多集中在画幅最边缘的无重叠区，可适当留出外边距拍摄。
  • 黑布 / 背景被当成植被？ 适当调高「近红外反射率下限」（如 0.15→0.25）通常有效，它与 NDVI 阈值是“与”关系。
  • 反射率偏高 / 偏低？ 检查灰板 / 定标板标称反射率是否填对，以及是否有饱和波段。
